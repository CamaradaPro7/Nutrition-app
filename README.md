# Nutrition App

Aplicación personal de nutrición y alimentación.

## Objetivo

Registrar las comidas diarias de forma sencilla desde móvil.

La aplicación será independiente de la aplicación de salud.

## Funciones V1

- Desayuno
- Comida
- Merienda
- Cena

- Registro de alimentos
- Historial diario
- Informes para ChatGPT
- Exportación JSON
- Importación JSON
